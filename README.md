# EVCompare
- Full webapp that calculates the EV-petrol savings and financing.
- The website finds the best value for the client, 'people first, not profit' is the motto.

### What will the webbapp do?
- Calculate the EV-Petrol savings.
- Calculate the Loan payments and take in the account the EV-Petrol savings.
- Collect leads (contact/fill out form).
- Route users to affiliated websites (Best value for the customer).
- Adjust the data, prices, offers to the country that the user is in (Localization).
- Suggest a car/financing -> Add budget (down payment, monthly payment) -> simplified version.
- Solar system amortization calculated into the EV savings.
